# Project conventions

## Authorship / attribution (strict)

- NEVER add any AI/assistant attribution to commits, pull requests, or any
  artifact pushed to this repository.
- Specifically, do NOT add:
  - `Co-Authored-By: Claude ...` trailers
  - `🤖 Generated with Claude Code` / `Generated by Claude Code` lines or links
  - any mention of "Claude", "Anthropic", or assistant session URLs
- All commits and PRs must be authored as the repository owner
  (Rayan <rayankr69@gmail.com>) — name and email only, nothing else.
- Do not use branch names containing "claude".
